# 🛒 Au - Champ ERP (Système de Gestion Local)

Ce projet est une solution de gestion de stock et de vente conçue pour les moyennes et grandes surfaces.

## 🛠️ Stack Technique
- **Interface** : Streamlit (Python)
- **Base de données** : MySQL (XAMPP)
- **Gestion de version** : Git / GitHub

## 📂 Installation Locale
1. Cloner le dépôt.
2. Installer les dépendances : `pip install -r requirements.txt`.
3. Importer `database.sql` dans votre serveur MySQL local.
4. Lancer : `streamlit run app.py`.
